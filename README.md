# Samurai Tap

## ゲーム内容
* 片手でサクッと遊べる新感覚ピンポンバトル
* 操作法やUIなど、ゲームとして成立しうるギリギリまで要素を削減した

* チュートリアルは実装する時間がなかったためWebサイトとして用意→Unity内でWebViewを使ってみる！
* 学習と平行して実装したため、設計思想に一貫性がなく自分でも修正が大変→要リファクタリング
